# Mechatronics-card-sorting-system
The Automated Pick-and-Place Card Sorting System is a fully integrated mechatronic machine capable of identifying and sorting a deck of playing cards into predefined bins. 
The system demonstrates core principles of industrial automated sorting process and reflects mechanisms commonly found in real-world applications, such as gantry cranes used in loading docks and arcade-style claw machines, but it is automated.

The key features of the system:
	1 - Programmable Logic Controller (PLC)
	2 - Three-axis gantry
	3 - Machine Vision (USB camera)
	4 - Pneumatic circuit
